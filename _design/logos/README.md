# Leland Logo Assets

PNG logo files for embedding in PPTX presentations.

## Required Files

| File | Description | Usage |
|------|-------------|-------|
| `leland-logo-white.png` | Full logo in white | Dark backgrounds (title slides, CTAs) |
| `leland-logo-green.png` | Full logo in Leland Green (#15B078) | Light backgrounds, footers |
| `leland-logo-gray.png` | Full logo in gray (#9B9B9B) | Subtle footer branding |
| `leland-compass-white.png` | Compass icon only in white | Dark backgrounds |
| `leland-compass-green.png` | Compass icon only in green | Light backgrounds |
| `leland-compass-gray.png` | Compass icon only in gray | Footer accents |

## Specifications

- **Format:** PNG with transparency
- **Resolution:** Export at 2x for crisp display (200px height for full logo, 50px for compass)
- **Color accuracy:** Use exact hex values from brand guidelines

## Export from SVG

The SVG source for logos is embedded in the HTML templates. To export:

1. Open any HTML template (e.g., `01-title-dark.html`)
2. Copy the SVG code for the logo
3. Use an SVG editor (Figma, Illustrator, Inkscape) to export as PNG
4. Ensure transparency is preserved

## Color Values

- White: `#FFFFFF`
- Leland Green: `#15B078`
- Gray: `#9B9B9B`
- Darkest Green: `#113A2D`
