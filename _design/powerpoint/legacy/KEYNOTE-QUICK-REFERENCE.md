## Keynote Shortcuts

| Action | Shortcut |
|--------|----------|
| Edit Master Slides | View > Edit Master Slides |
| Add Text Box | T |
| Add Shape | S |
| Group Objects | Cmd + G |

| Define Custom Color | Format > Advanced > Define Custom Colors |

**Dark slides:**
- Left: White Leland logo

## Common Element Styles

**Cards:**
- Background: White
- Border: 1px #E5E5E5
- Corner radius: 12px
- Padding: 20px

**Numbered Steps:**
- Circle: 24-32px, #15B078
- Number: White, centered
- Connector: 2px #E5E5E5

**Buttons/CTA:**
- Background: #15B078
- Text: White, Semibold
- Corner radius: 8px
- Padding: 16px 32px

**Eyebrow Labels:**
- Color: #15B078
- Size: 13pt
- Weight: Semibold
- Style: UPPERCASE
- Letter spacing: 0.1em
