# Slide Content Rules

> **Global reference:** For complete design rules (punctuation, icon colors, card grids, workflow alignment, sentence count), see `production-rules.md`, `css-reference.md`, and `qa-rules.md` (all in `_design/revealjs/`). This file contains only MBA-specific rules.

---

## 1. School References (MBA-Specific)

- **Avoid** naming specific schools in general advice slides. Use tier groupings (M7, T15) instead.
- School names **are fine** in example/case-study slides, speaker notes, or when illustrating a specific data point.
- Use the standard tier groupings:
  - **M7**: Harvard, Stanford, Wharton, Columbia, Booth, Kellogg, Sloan
  - **T15**: Haas, Yale SOM, Fuqua, Tuck, Ross, Darden, Stern, Anderson
